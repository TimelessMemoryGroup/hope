# Upload Folder

This folder contains the website files that will be deployed to Cloudflare Pages.

## Contents

- `index.html` - The main homepage of the website

## Adding New Pages

Simply add new HTML, CSS, JavaScript, or other static files to this folder. They will be automatically deployed when pushed to the main branch.

## Structure

```
upload/
├── index.html          # Homepage
├── css/               # (optional) CSS files
├── js/                # (optional) JavaScript files
├── images/            # (optional) Images
└── ...                # Other static assets
```
